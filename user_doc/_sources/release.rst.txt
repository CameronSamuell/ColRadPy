Release Notes
=============

Summary
-------
ColRadPy 0.9 is a beta relase of the ColRadPy code.
ColRadPy is a collisional radiative solver as well as an ionization balance solver.
For background theory and example of ColRadPy's use see upcoming Johnson Nuclear materials paper.

Installation Instructions
-------------------------

Right now ColRadPy has been tested with Anaconda installed python3.
